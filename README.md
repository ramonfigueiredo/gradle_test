# Simple project using Gradle Build Tool

## Gradle Tasks

Displays the tasks runnable from root project 'gradle_test'.

```
./gradlew tasks
```

## Gradle Build

Assembles and tests this project.

```
./gradlew build
```

## Gradle Run Tests

Runs the test suite.

``` 
./gradlew build
```

## Running the application

### English

Run:

```
java -jar build/libs/gradle_test.jar en
```

Output:

